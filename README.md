# Decision3

**Infrastructure for the autonomous AI economy.**

---

## What We Build

Three things autonomous AI agents need to operate in the real world:

1. **Pay** across diverging payment standards — without vendor lock-in
2. **Remember** inside disciplined context boundaries — without hallucination drift  
3. **Identify** as named individuals — not anonymous corporations

None existed as production-grade, open tools. So we built them.

---

## Products

### Interouter — Pillar 1 (LIVE)
Facilitator-agnostic middleware for the x402 payment standard.

- Cross-facilitator compatibility (OpenLedger + OpenGradient)
- 5-stage payment lifecycle with fallback routing
- Circuit breaker protection
- v0.1.6 live on npm · 45/45 tests passing · MIT licensed

npm install @decision3/interouter-core
